# BarakahHub - Presentasi Aplikasi

## Apa itu BarakahHub?

**BarakahHub** adalah platform digital berbahasa Indonesia yang membantu Masjid kecil dalam membuat halaman program Ramadan dengan konten yang dihasilkan oleh AI dan menerima donasi secara online.

---

## Masalah yang Diatasi

- Masjid kecil kesulitan membuat materi promosi program Ramadan
- Proses pengumpulan donasi masih manual dan tidak efisien
- Kurangnya sumber daya teknis untuk membuat halaman web

---

## Fitur Utama

### 1. Pembuatan Program Otomatis dengan AI
- Konten program Ramadan dihasilkan secara otomatis oleh AI
- Deskripsi program, teks WhatsApp, dan caption Instagram dalam Bahasa Indonesia
- Hemat waktu dan biaya promosi

### 2. Penerimaan Donasi Online
- Integrasi dengan Mayar untuk pembayaran donasi
- QR Code otomatis untuk memudahkan donatur
- Kampanye donasi dapat dibuat dengan mudah

### 3. Halaman Publik Masjid
- Setiap Masjid mendapatkan halaman publik sendiri (`/m/[slug]`)
- Tampilan responsif dan profesional
- Tanpa perlu keahlian teknis

### 4. Dashboard Pengelola
- Kelola program Ramadan dengan mudah
- Pantau progress donasi
- Analitik sederhana

---

## Teknologi yang Digunakan

- **Frontend:** Next.js 16, React 19, TypeScript, TailwindCSS
- **Database:** SQLite (Drizzle ORM)
- **AI:** Sumopod (GPT-4o-mini)
- **Payments:** Mayar API
- **Auth:** NextAuth v5
